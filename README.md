# Canal de Slack

Canal de Slack para utlizar en Botpress.

### Prerequisitos
 - node v10
 - yarn
 
### Instalación

    yarn build

### Comandos disponibles

#### `yarn start`

Inicia botpress.

#### `yarn start`

### Permisos
Se utilizan los siguientes permisos de Slack:
 - `users:read`: permite obtener información general acerca de la persona que está hablando con el bot.
